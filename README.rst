Installation
================================================================================

You can install it via pip:

.. code-block:: bash

    $ pip install robotchef_v2


or clone it and install it:

.. code-block:: bash

    $ git clone http://github.com/chfw/robotchef_v2.git
    $ cd robotchef_v2
    $ python setup.py install
